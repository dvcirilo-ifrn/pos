# Elementos
- `<!ELEMENT nome (tipos-de-dados)>`
- Tipos de dados:
    - `#PCDATA`
    - EMPTY
    - Sub-elementos na forma de lista:
        - `(sub-elem1, sub-elem2)`
        - `(sub-elem1|sub-elem2)`
        - `((sub-elem1|sub-elem2), sub-elem3)`
        - `(#PCDATA|sub-elem1|sub-elem2)`
- `|` - operador "ou"
- A ordem dos sub-elementos importa!
